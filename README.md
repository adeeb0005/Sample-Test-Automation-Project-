# Automation Testing Framework (Java + Selenium + RestAssured)

## 🔹 Overview
This project is a hybrid automation framework showcasing:
- API Testing (RestAssured)
- UI Testing (Selenium)
- Database Testing (MSSQL)
- End-to-End Integration Testing

## 🔹 Test Coverage
- 110+ API test cases
- 60+ UI test cases
- UI + API + DB integration tests

## 🔹 Tech Stack
- Java
- Selenium WebDriver
- RestAssured
- TestNG
- Maven
- SQL Server

## 🔹 Features
- Modular framework design
- Reusable utilities
- API + UI + DB validation
- CI/CD ready structure

## 🔹 Purpose of This Repository
This repository is intended for **code structure and framework design showcase only**.

Due to confidentiality and NDA restrictions:
- Real API endpoints have been removed
- Database connections are not active
- Test data is sanitized

## 🔹 Execution Note
This project is **not runnable in its current form**.  
It serves as a demonstration of how a real-world automation framework is designed and structured.
